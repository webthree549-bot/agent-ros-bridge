## New Features (v0.4.0)

### 1. MCP Server Module

**File:** `agent_ros_bridge/mcp_server.py`

**Capabilities:**
- Auto-discover ROS actions → MCP tools
- Auto-discover ROS topics → MCP resources  
- JWT authentication integration
- Claude Desktop compatible

**Usage:**
```bash
python -m agent_ros_bridge.mcp_server
```

**Config (claude_desktop_config.json):**
```json
{
  "mcpServers": {
    "ros": {
      "command": "python3",
      "args": ["-m", "agent_ros_bridge.mcp_server"],
      "env": {"JWT_SECRET": "..."}
    }
  }
}
```

## Key Decisions

1. **MCP as addition, not replacement**
   - Keep WebSocket/gRPC for power users
   - MCP for ease of use

2. **Auto-discovery default**
   - Automatically expose ROS actions/topics
   - Configurable allow/deny lists

3. **Security maintained**
   - JWT required even for MCP
   - No bypass or mock modes

4. **Examples show both**
   - Side-by-side comparison
   - User chooses their path
