get_openclaw_adapter(include_ros1: bool = False)[source]¶

Get OpenClaw adapter for this bridge.

Provides integration with the OpenClaw AI agent framework through ClawHub skills. The adapter provides:

  1. Skill Mode: Access to ClawHub skill for OpenClaw agents

  2. Extension Mode: Direct tool execution (optional)

Example

adapter = bridge.get_openclaw_adapter()

# Get skill path for ClawHub skill_path = adapter.get_skill_path()

# Or use extension mode tools = adapter.get_tools() result = await adapter.execute_tool(“ros2_publish”, {…})

Parameters:

include_ros1 – Whether to include ROS1-specific tools.

Returns:

OpenClawAdapter instance.
