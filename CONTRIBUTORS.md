## How to Add Yourself

Contributed? Add yourself with:

```bash
# Using all-contributors CLI
npx all-contributors add <username> <contributions>
```

Contribution types:
- `code` - Code contributions
- `doc` - Documentation
- `test` - Tests
- `bug` - Bug reports
- `ideas` - Ideas & planning
- `example` - Examples
